1. Make a github repo and clone into our system using powershell
2. Creating data and dataframe in mycode file
3. Git add and commit and push
4. Installing dvc by pip install dvc
5. Initializing dvc by dvc init command
6. Now do "mkdir S3" (create a new S3 directory)
7. Giving the remote path of the data storage in our case is S3 but can be URL of AWS cloud using command "dvc remote add -d myremote S3"
8. We want that our data that is in data folder should be tracked by dvc ("dvc add data/")
